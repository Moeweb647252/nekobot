# Nekobot Script Middleware

## Description

This crate provides a middleware for Nekobot that allows agents to execute JavaScript code using the Boa engine with embedded nekobot internal tools. It enables agents to run JavaScript code in a sand-boxed environment, allowing for dynamic behavior and interaction with the Nekobot ecosystem.

## Features

- Channel Integration: Provides an interface for agents to interact with various communication channels supported by Nekobot.
