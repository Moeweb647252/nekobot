# Nekobot core crate

This crate provides the core functionality for the Nekobot ecosystem. It defines the core traits, structures, and interfaces that are used by other crates in the ecosystem to implement various features and functionalities. The core crate serves as the foundation for building agents, channels, providers, skills, and other components that make up the Nekobot system. It also provides utilities and common functionality that can be shared across different parts of the ecosystem, ensuring consistency and ease of development.
